# Button Atom component

Button Atom component.

## Source

```js
<Button>Button content</Button>
```

## Props

- `children` - (required) The content of the rendered Button.
- `className` - (optional) Any extra classes to add to the rendered Button.
- `href` - (optional) Adds the `href` attribute and automatically converts the button element to a link element.
- `isDisabled` - (optional) Adds the `disabled` attribute and `--disabled` modifier.
- `isFullWidth` - (optional) Adds the `--full-width` modifier.
- `isHollow` - (optional) Adds the `--hollow` modifier.
- `isUnstyled` - (optional) Adds the `--unstyled` modifier.
- `tabIndex` - (optional) Adds the `tabindex` attribute.
